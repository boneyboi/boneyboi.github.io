    Projects

It's a simple wordpress website about the anxiety and stress relieving benefits of Coloring Books! I use a script to prompt ChatGPT to generate articles and content for the website for me. This website will continue to post articles until the end of the year.

    Magic the Gathering competitive AI

    One of my research projects at Furman Universty. I worked with a team to make an AI that could play Magic the Gathering on a software simulator called "Forge". I was responsible for making the data structures for the project. We used Java to make this AI and we did successfully make an AI that was better than the current AI's that play against you in-game.

    Speedrunning notes

    Another research project where a mentor and I made a component of the LiveSplit program. This component allowed the user to write and save notes of text, images, and custom emojis. These notes would run along side each split of the game that a person was playing to give them dynamic note displays relevant to the part of the game they are in.

    Additionally, I was the first person to properly research and build a component for LiveSplit in C# and published a template for people to start making their own components!

    Video game Prototype

    During my Game design class a team and I made a fighting game prototype based on music and rythm. The player would input moves on the beat to create devastating rythm based combos! I was the director/programmer and led the team by creating designs goals and mechanics.

    Unity VFX project

    Lastly, I've dabbled in making shaders for objects in Unity. Utilizing some C# and HLSL to make interesting artistic renders on models.
